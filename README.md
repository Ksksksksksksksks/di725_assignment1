# Transformer-Based Sentiment Classification forCustomer Service Conversations

This repository contains code for fine-tuning a transformer-based model to classify customer sentiment from service conversations. The model combines **DistilBERT** text encoder with categorical features (issue area, issue category, product sub-category) to predict three-class sentiment (negative, neutral, positive).

Despite severe class imbalance, the model achieves **70% test accuracy**, with strong performance on negative and neutral classes. Key experiments tracked in [Weights & Biases](https://wandb.ai/e278979-metu-middle-east-technical-university/odtu-transformer-assignment/workspace?nw=tka5shqmrc9) include comparisons of pooling strategies, categorical feature integration methods, and class imbalance handling techniques.
